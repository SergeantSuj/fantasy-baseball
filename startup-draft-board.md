# Startup Draft Board

This is the opening seed board for the 2026 startup draft simulation.

It is not meant to be an immutable overall ranking. It is the initial board before manager-specific preferences, injury updates, ADP timing, and roster construction adjustments move individual picks.

## Round 1 Rules

Default first-round priorities:

1. elite five-category or OBP-anchor bats
2. elite young franchise bats
3. only the very top ace tier among pitchers
4. do not force prospects in Round 1

## Notes On The Board

### OBP Bumps

These players gain extra value in this league format:

- Juan Soto
- Yordan Alvarez
- Bryce Harper
- Gunnar Henderson
- Vladimir Guerrero Jr.

### Speed Scarcity Bumps

These players can change roster shape early:

- Bobby Witt Jr.
- Elly De La Cruz
- Julio Rodriguez
- Ronald Acuna Jr.
- CJ Abrams

### Round 1 Pitching Exception List

Only these pitchers should normally threaten the first round in a neutral dynasty room:

- Tarik Skubal
- Paul Skenes

Everyone else should usually be treated as a Round 2 or later play unless a manager profile is explicitly ace-driven.

## First-Round Pick Logic By Slot

### Picks 1-3

- take the best elite bat left
- only take Ohtani over a pure hitter if the room treats him as a true top-three dynasty asset

### Picks 4-7

- prioritize OBP-safe stars and category anchors
- do not overthink age discounts on clear first-round bats

### Picks 8-10

- watch for the last Tier 1 bat
- if Tier 1 is gone, pivot to the best OBP-safe slugger or rare speed/power combo
- only take Skubal or Skenes here if the manager profile strongly prefers ace stability

## How This Board Should Be Used

1. Start from these tiers.
2. Apply manager personality.
3. Apply roster-state logic once the draft is underway.
4. Let ADP shape timing, not overall philosophy.

If a simulated pick comes from outside the current tier, the selection should still be explainable by one of these reasons:

- category need
- age preference
- injury discount
- prospect aggression
- pitcher aggression
